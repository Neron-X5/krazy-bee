Reactjs Test

We are going to use a couple of JSON APIs for this test. The first API is used to get a list of albums. The second API is used to get a list of photos from each album. The json APIs are described below.

Album Listing API

API: https://jsonplaceholder.typicode.com/albums

Doing a GET request will give a JSON output as shown below. 

[
  {
    "userId": 1,
    "id": 1,
    "title": "quidem molestiae enim"
  },
  {
    "userId": 1,
    "id": 2,
    "title": "sunt qui excepturi placeat culpa"
  }
]

You will need to parse the json and have access to title, id and userId. title (album title) and userId are used in the header section as shown in output section.

Album photos listing API

For each of the albums, you need to make another API call with the id given to albumId parameter in the API as shown.

API: https://jsonplaceholder.typicode.com/photos?albumId=1

Doing a GET request will give a JSON output as shown below.

[
  {
    "albumId": 1,
    "id": 1,
    "title": "accusamus beatae ad facilis cum similique qui sunt",
    "url": "https://via.placeholder.com/600/92c952",
    "thumbnailUrl": "https://via.placeholder.com/150/92c952"
  },
  {
    "albumId": 1,
    "id": 2,
    "title": "reprehenderit est deserunt velit ipsam",
    "url": "https://via.placeholder.com/600/771796",
    "thumbnailUrl": "https://via.placeholder.com/150/771796"
  }
]

You will need to parse the json and have access to id, title and url. These three are used in the card section as shown in the output below. Image from url is shown as different color boxes. title is shown below image. id is shown below title.

Output

Each row is for an album. The album needs to be lazy loaded as we scroll down. 

Each album (row) is a carousel. As we scroll right or left using the buttons, the images need to be lazy loaded.
